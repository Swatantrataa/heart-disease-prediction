# Heart Disease Analysis & Prediction

This project analyzes heart disease data using Microsoft Excel and presents insights through an interactive Power BI dashboard. It also includes basic Python code for prediction.

## Features
- Data cleaning and preparation
- Data analysis using Excel
- Interactive dashboard using Power BI
- Basic prediction using Python

## Tools Used
- Microsoft Excel
- Power BI
- Python

## Dashboard
The project includes a Power BI dashboard to visualize key insights and trends related to heart disease.

## Additional Work
Basic Python code has been included for heart disease prediction.
